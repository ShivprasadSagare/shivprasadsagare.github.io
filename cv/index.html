Publications

  •
    XWikiGen: Cross-lingual Summarization for Encyclopedic Text Generation in Low Resource Languages
    The 2023 ACM Web Conference
    We propose a novel task of generating Wikipedia article section text in low-resource languages by cross-lingual summarization of multiple reference articles. Using Wikimedia APIs, we build a dataset XWikiRef, which consists of section text paired with section-specific citations in 15 languages and across eight domains. We implement a 2-stage approach, i.e., extractive followed by abstractive summarization. We experiment with saliency-based and graph-based methods for the extractive stage to filter the relevant sentences. We analyze the performance of pretrained language models mT5 and mBART in the abstractive stage using diverse evaluation metrics.
  •
    XF2T: Cross-lingual Fact-to-Text Generation for Low-Resource Languages
    arXiv
    We conduct an extensive study using popular Transformer-based text generation models on our extended multi-lingual dataset. Further, we investigate the performance of different text generation strategies: multiple variations of pretraining, fact-aware embeddings, and structure-aware input encoding. Our extensive experiments show that a multi-lingual mT5 model which uses fact-aware embeddings with structure-aware input encoding leads to the best results on average across the twelve languages.
  •
    XAlign: Cross-lingual Fact-to-Text Alignment and Generation for Low-Resource Languages
    The Web Conference 2022, ACM
    In this work, we propose a novel task of cross-lingual fact-to-text generation. We specifically build a parallel dataset consisting of English Wikidata facts aligned with natural language sentences across 8 Indian languages. This dataset will serve as a benchmark to build better generative models for low-resource languages.
  •
    Cross-lingual Alignment of Knowledge Graph Triples with Sentences
    ICON 2021: 18th International Conference on Natural Language Processing
    In this paper, we address the unsupervised pairing of knowledge graph triples with sentences for low-resource languages, selecting Hindi as the low-resource language. We propose two novel approaches: NER-based filtering with Semantic Similarity and Key-phrase Extraction with Relevance Ranking. We publically make available a collection of 29224 well-aligned English triples and Hindi sentence pairs.
